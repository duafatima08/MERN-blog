Prerequisites

Before webegin, ensure you have the following installed:

  • Node.js (LTS version recommended)
  • MangoDB (local or cloud-based via MangoDB Atlas)
  • npm or yarn (for package managment)
  • Visual Studio Code (or any preferred code editor)

Step 1: Setting Up the Project

Initialize the Backend (Express and MongoDB)

1. Create a new project folder:

mkdir mern-app && cd mern-app

2. Initialize a Node.js project:

npm init -y

3. Install necessary backend dependencies:

npm install express mongoose cors dotenv

4. Install development dependencies:

npm install --save-dev nodemon @types/node @types/express

5. Create the required folders:

mkdir server && cd server
mkdir models routes config

Setup Express Server
Create an index.ts file inside the server folder:

import express from 'express';
import cors from 'cors';
import mongoose from 'mongoose';
import dotenv from 'dotenv';

dotenv.config();

const app = express(); const PORT = process.env.PORT || 5000;

app.use(cors());
app.use (express.json());

.then(()=> console.log ('MOngoDB'))
.catch(err => consol.error(err));

app.listen(PORT, () => console.log('Server running on port ${PORT}'));

Configure .env File
Create a.env file in the root directory:

MONGO_URL=your_mongodb_connection_string
PORT=5000

Step 2: Setting Up the Frontend (React + TypeScript + Tailwind CSS)

Initialize React App:

npx create-react-app client --template typescript

Install Tailwind CSS:

cd client
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

Configure Tailwind (tailwind.config.js):

module.exports = {
  content: ['./src/**/*.{js,ts,jsx,tsx}'],
  theme: { extend: {} },
  plugins: [],
};

Import Tailwind (index.css):

@tailwind base;
@tailwind components;
@tailwind utilities;

Step 3: Building the UI Components

Create src/components/Button.tsx:

import React from 'react';

type ButtonProps = { text: string };

const Button: React.FC<ButtonProps> = ({ text }) => {
  return (
    <button className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600">
      {text}
    </button>
  );
};

export default Button;

Step 4: Connecting the Frontend and Backend

Modify src/App.tsx:

import React, { useEffect, useState } from 'react';

const App: React.FC = () => {
  const [message, setMessage] = useState('');

  useEffect(() => {
    fetch('http://localhost:5000')
      .then(res => res.json())
      .then(data => setMessage(data.message));
  }, []);

  return (
    <div className="flex justify-center items-center h-screen bg-gray-100">
      <h1 className="text-xl font-bold">{message}</h1>
    </div>
  );
};

export default App;

Step 5: Running the Mern Application

Start the backend server:

cd server
nodemon index.ts

Start the fronted application:

cd client
npm start

Your MERN stack applicationis now running with Typescript and Tailwind CSS, offering a
robust development experience.

Conclusion

By following this guide, you have successfully set up a MERN stack application with Tailwind
CSS and TypeScript. This stack provides a powerful foundation for scalable and efficient web applications.
You can extend this setup by adding authentication, integrating APIs, and
enhancing Ul components with Tailwind's utility classes. Happy coding!
